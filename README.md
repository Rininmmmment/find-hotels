# find-hotels
楽天トラベルに掲載されたホテルを探すことができるアプリです。<br>
https://osushicat.sakura.ne.jp/

## 使用したもの
- Vue.js + Vite
- tailwindCSS / daisyUI
- Google Custom Search
- 楽天トラベル施設検索API
- HeartRails Express API
- さくらのレンタルサーバー

## 追加したい機能
- 駅からの半径を選択
- マップを表示し、駅からの半径x[km]のときの円を表示
- 駅名を入力し、Enterを押すと該当する駅名が選択できるようになる
